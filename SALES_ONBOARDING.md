## 5. The Outreach Playbook

### The Goal of Every First Message
Get them to sign up for the free tier. Not to buy. Not to book a call. Just: **try it**.

Once they have an API key and 10,000 tokens, the product sells itself better than any message can.

### Message Formula (LinkedIn / Email / DM)
```
Subject / opening: [Specific thing about what they're building]

Body:
Saw you're building [X]. Afferens might be relevant — it's a REST API that gives
AI agents real-time physical perception: vision, GPS, temperature, gas detection,
sound events. Free tier, 10,000 tokens, no card required.

One curl call to try it: afferens.com/api/demo?modality=VISION

Happy to answer any questions if useful.

[Your name]
```

**Rules:**
- Personalise the opening line — reference their actual project or company
- Do NOT pitch features. Offer the free trial.
- Keep it under 100 words
- No emojis
- No fabricated metrics ("used by 10,000 developers" — we don't have that yet)
- Never claim capabilities we don't have

### Follow-up (if no reply after 3 days)
```
Hey [name] — just checking if you had a chance to try the demo.
afferens.com/api/demo?modality=VISION — takes 30 seconds.
Happy to help if you hit any issues.
```

One follow-up only. Don't spam.

## 6. Converting Free to Paid

When someone signs up, they get 10,000 free tokens. Once they start using the API, tokens go down. Your job is to be there when they run low.

**Trigger message (when they're actively using it):**
```
Hey [name] — noticed you've been using the API. If you're running low on tokens,
the Builder pack (200K tokens) is $29. afferens.com/pricing
Happy to answer anything if you're evaluating whether to go deeper.
```

Faris can see token usage in the admin panel — ask him to flag any free-tier users burning tokens fast. Those are your warmest leads.
